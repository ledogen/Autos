## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-07-01-DOS | Denial of Service | freecam mousemove/keydown listeners | accept | Listeners do O(1) work; mousemove only mutates two floats. No remote attacker; single-origin static page. |
| T-07-01-PL | Tampering | pointer-lock state | mitigate | `isPointerLocked` is derived solely from `document.pointerLockElement` via the browser `pointerlockchange` event — never set speculatively; mouse-look gated on it so a lost lock cannot drive the camera. |

No HIGH-severity threats — static client-side app; this plan introduces no network, storage, or untrusted-string input.
