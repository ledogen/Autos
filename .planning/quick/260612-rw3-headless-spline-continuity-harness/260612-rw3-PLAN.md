<objective>
Build a headless Node ESM harness that numerically measures spline geometry continuity — the verification gate for upcoming road-spline (Phase 8 routing) work.

Purpose: Today the only way to know whether a graded road spline is smooth is to run WebGL and eyeball it. This harness lets the agent self-verify continuity numerically. Root cause being targeted: the spline's control-point Y = raw terrain + 1 m, so curvature/grade spike and the per-tile render-time smoothing band-aid creates seam steps. The eventual fix bakes a continuous, curvature-limited graded Y into the spline; this harness is the GATE that proves it worked.

Output: `test/spline-continuity.mjs` — plain Node ESM, zero install, importing only the THREE-free pure helpers from `src/road-carve.js`. Runs as `node test/spline-continuity.mjs`, prints a metric table, exits nonzero when a real gate is exceeded.
</objective>

  - tangentAt(t): first derivative direction in the XZ plane, returned as a UNIT {x,z} (normalize; if degenerate return {x:1,z:0}). This feeds signedCurvature. A central finite-difference of getPoint at t±h (h≈1e-4, projected to XZ then normalized) is acceptable and simplest — document the choice.
  - getLength(): approximate arc length by summing chord lengths over a dense sampling (e.g. 200 subsamples). Used to convert SAMPLE_INTERVAL_M into a sample count.

<task type="auto">
  <name>Task 2: Metrics, PASS/FAIL table, and exit-code gate</name>
  <files>test/spline-continuity.mjs</files>

Add the metric computation, reporting, and gate logic to test/spline-continuity.mjs (same file, append below Task 1 content).

For each fixture: build the curve via the vendored sampler, compute N = max(2, ceil(getLength()/SAMPLE_INTERVAL_M)) samples at uniform t = i/(N-1). At each sample collect: position {x,y,z}, XZ unit tangent (tangentAt), and arc-length position arcS (running sum of 3D chord lengths between consecutive samples). Then compute metrics over adjacent sample pairs:
  - maxVStep = max over i of |y[i+1] - y[i]|  (vertical step, metres).
  - kappa[i] = signedCurvature(T[i].x, T[i].z, T[i+1].x, T[i+1].z, ds) where ds = arcS[i+1]-arcS[i] (guard ds>1e-9). This yields a per-pair curvature series.
  - maxDKappa = max over i of |kappa[i+1] - kappa[i]| / max(ds, 1e-9)  (curvature rate, 1/m per m).
  - camberDeg[i] = clamp(CAMBER_STRENGTH * kappa[i], -CAMBER_CLAMP_DEG_RAD, +CAMBER_CLAMP_DEG_RAD) converted to degrees (multiply rad by 180/PI). Use the clamp in radians then convert, OR clamp directly in degrees — document which. maxDCamber = max over i of |camberDeg[i+1] - camberDeg[i]| / max(ds,1e-9)  (deg per metre).
  - boundaryMismatch: only for the seam fixture (role-driven or presence of sliceA/sliceB / boundaryIndex). Compute |y_endA - y_startB| at the shared boundary. For non-seam fixtures report N/A (e.g. null / '—').
